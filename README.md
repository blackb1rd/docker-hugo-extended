[Hugo](https://gohugo.io/) is a static site generator, written in Go.

This Docker Image is based on Alpine and included with
- hugo-extended
- rsync
- node with yarn
- mozjpeg (/opt/mozjpeg/)
- pngquant (/opt/pngquant/)
